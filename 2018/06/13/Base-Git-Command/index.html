Base Git command

Git 指令操作。

mkdir 创建一个空目录

git init 初始化目录

git add 把文件添加到仓库

git commit 提交文件到仓库

git status 查看仓库当前状态

git diff 查看文件修改内容

git log 查看版本历史纪录

git reset 版本回退

git reflog 查询记录的每一次命令
